Quick start

From ISO to PlasmaTV in a few minutes.

1
Download the ISO and write it to a USB drive: sudo dd if=plasmatv-linux.iso of=/dev/sdX bs=4M status=progress
2
Boot from the USB. The installer starts automatically on tty1.
3
Follow the dialog prompts — pick a disk, set up networking, and confirm the install.
4
Reboot, complete the TV OOBE wizard (language, Wi-Fi, theme), and enjoy PlasmaTV.
