Building the ISO

Build your own PlasmaTV Linux ISO from source.

Prerequisites

  • An Arch Linux system (the build uses mkarchiso which is Arch-specific)
  • archiso package installed: sudo pacman -S archiso
  • About 10 GB of free disk space for the build artifacts

Build steps

1
Clone the repo:
git clone https://github.com/plasmatvlinux/plasmatvlinux.github.io.git
2
Run the build script:
cd PlasmaTV-Linux && sudo ./installer/build.sh
3
Find the ISO in out/:

Done. The ISO is at out/plasmatv-linux-YYYY.MM.DD-x86_64.iso.

How it works

The build script:

  1. Copies the package list (archiso/packages.x86_64) into the airootfs so the installer can use it for pacstrap
  2. Copies the skel/ tree (target system files) into the airootfs
  3. Runs mkarchiso with the custom archiso profile

The archiso profile lives in archiso/ and is based on the official releng profile with modifications for:

  • Plymouth boot splash (spinner theme)
  • Root auto-login on tty1 via getty override
  • Auto-launch of the dialog-based installer
  • Custom bootloader configs (single "System0" entry for BIOS/GRUB, UKI for UEFI)

Customizing the build

To add or remove packages, edit archiso/packages.x86_64. This file is used both for the live environment and for the target system's pacstrap.

To modify the installer, edit archiso/airootfs/root/install.sh.

To add or change system files that get copied to the target, edit files in skel/.

Note: The work/ directory contains root-owned build artifacts. Clean it with sudo rm -rf work out before rebuilding.
