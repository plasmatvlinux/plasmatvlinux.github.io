Features

Everything PlasmaTV Linux offers out of the box.

Plasma Bigscreen

The default session is KDE Plasma Bigscreen — a TV-optimized 10-foot UI designed for couch use. Navigate with a keyboard, mouse, or remote control. Big, readable cards, minimal UI chrome, and full focus on content.

One-click desktop switch

Two desktop files let you switch between Bigscreen and the classic Plasma Desktop: "Switch to Desktop" and "Switch to Bigscreen". Selecting one updates the SDDM autologin config and restarts the display manager — no command line needed.

User selector

Instead of a traditional login prompt, PlasmaTV boots into a custom user selector screen. Select your profile and, if configured, enter a PIN. This is the same flow used by streaming platforms like Netflix. The selector runs as a dedicated selector-user with minimal privileges.

25 streaming apps

Each streaming service runs as its own Electron kiosk app — a frameless, chromeless window that feels like a native TV app. Press Escape to quit back to Bigscreen. See the full list.

Parental controls

TV OOBE wizard

First boot launches a TV-style out-of-box experience that guides you through language, timezone, Wi-Fi, and theme selection. Clean, minimal, and designed for a TV screen.

Plymouth boot splash

Boots with Plymouth (spinner theme) and a quiet splash kernel command line — no verbose kernel messages, just a clean boot animation.

Auto-login, no prompts

SDDM auto-logs into the user selector. Extra TTYs (tty2-6) are masked — there's no login prompt reachable except through the graphical session. Root login is locked entirely.

Audio

Uses PipeWire + WirePlumber — the modern PulseAudio-compatible stack that ships on fresh Arch/KDE installs. No PulseAudio dependency.

Rolling releases

Based on Arch Linux — once installed, sudo pacman -Syu keeps everything up to date. No need to wait for a new ISO.
