Installation guide

From ISO to PlasmaTV in about 10 minutes.

1
Download an ISO — you have two options:
  • Netinstall ISO (69 MB) — tiny, downloads packages over the internet during installation. Needs a wired Ethernet or Wi-Fi connection.
  • Full ISO (2.1 GB) — includes everything. Larger download, no internet needed during install.
2
Write it to a USB drive. Find your USB device with lsblk, then:
sudo dd if=plasmatv-linux-*.iso of=/dev/sdX bs=4M status=progress conv=fsync
Replace /dev/sdX with your actual device (not a partition).
3
Boot from the USB. On most machines, press F12 during boot to select the boot device, or configure the BIOS to boot from USB first.
4
The installer starts automatically. Root auto-logs in on tty1 and launches the dialog-based installer. Follow the on-screen prompts.
5
Select a disk. The installer will ask you to pick a disk. Everything on it will be erased.
6
Set up networking — ethernet is detected automatically; Wi-Fi can be configured interactively if no wired link is found.
7
Wait for the install. pacstrap downloads and installs all packages. This takes 5-15 minutes depending on your internet speed.
8
Reboot. Remove the USB drive and reboot. The system boots into the PlasmaTV user selector.
9
Complete the OOBE. On first boot, the TV OOBE wizard walks you through language, timezone, Wi-Fi, and theme setup.

Note: The 64-bit UEFI path uses a UKI registered directly with efibootmgr. If Secure Boot is enabled, either disable it or sign the UKI yourself.

Converting an existing Arch install

Already running Arch and don't want to reinstall? Clone the repo and run the transform script:

git clone https://github.com/plasmatvlinux/plasmatvlinux.github.io.git
cd PlasmaTV-Linux
sudo ./transform-existing-system.sh

The script installs the PlasmaTV package set, creates tv-user/selector-user, wires up SDDM autologin, and enables the PlasmaTV services. It does NOT touch your disk layout, existing users, or personal files.
